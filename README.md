# Service Template

Template for creating web services in Go.

## Features

- [x]  Config
- [x]  Graceful Shutdown
- [x]  Liveness handler
- [x]  Authentication
- [x]  JWT generation
- [x]  Request parsing
- [x]  Response handling
- [x]  Logging
- [x]  Metrics
- [x]  Panic handling
- [x]  Error handling
- [x]  Docker support
- [x]  Kubernetes support
- [x]  Postgres connection
- [x]  Migrations
- [x]  Tracing
