- [x]  Graceful shutdown
- [x]  Postgres connection
- [x]  Seeding
- [x]  CRUD functionality
- [x]  Validation

- [ ]  Manejo de errores
- [ ]  Logging MW
- [ ]  Health checks
- [ ]  Autenticación
- [ ]  Autorización
- [ ]  Mensaje de error de validación
- [ ]  K8s
- [ ]  Tracing
- [ ]  Métricas
- [ ]  CI/CD
